About

We encourage open source software development.

Free and open source software (FOSS) is software developed by informal collaborative networks of programmers. The source code is licensed free of charge, encouraging modifications and improvements. FOSS (free and open-source software) allows the user to inspect the source code and provides a high level of control of the software's functions compared to proprietary software. Open-source software does what it says on the tin - the source code of a piece of software is kept open and free to download, modify and incorporate into third party projects, which helps to improve the software itself over time.

Our purpose is to deliver new ideas, new methods, new approaches, new inventions, new materials and/or new applications that drive value. Creativity, the ability to generate novel and useful ideas, is the seed of innovation but unless it’s applied and scaled it’s still just an idea.
